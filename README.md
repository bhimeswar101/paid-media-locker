# Paid Media Locker

A secure full-stack paid media locker application built with Node.js, Express, PostgreSQL, Prisma, and React Native (Expo).

## Tech Stack

### Backend
- Node.js
- Express
- TypeScript
- Prisma
- PostgreSQL

### Mobile
- React Native
- Expo

## Features

- JWT Authentication
- Virtual Wallet
- Paid Image Upload
- Preview Generation
- Secure Media Unlock
- Transaction History

> This project is being developed as part of a Full Stack Developer Internship assignment.
